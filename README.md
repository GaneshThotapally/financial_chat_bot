An AI-powered Personal Finance Dashboard built using Streamlit, IBM Granite LLM, OpenCV, OCR, and Plotly for smart financial assistance, tax calculations, budget analysis, and investment guidance.

---

## 📌 Overview

This project is an intelligent AI Finance Assistant that helps users with:

- Budget Analysis
- Tax Calculations
- Investment Suggestions
- Expense Tracking
- Savings Insights
- PDF & Image Financial Document Analysis

The system uses IBM Granite 7B LLM for generating professional financial advice in real-time.
